# 풋살장 예약 시스템

**프로젝트기간** : 2019.12.01 ~ 2020.01.01  
**사용스택** : Java, Mysql, Json  
**기획의도** : MVC패턴을 학습한 다음 적용해 보고자 시작한 프로젝트입니다.  
**프로젝트 내용**  
사용자는 풋살장을 예약 할 수 있으며 풋살장 관리자는 
풋살장 예약을 관리할 수 있는 풋살장 예약 시스템 입니다.  
사용자 모드와 관리자 모드(클라이언트)와 서버(DB)로 구성되어있는
클라이언트 서버 구조입니다.
